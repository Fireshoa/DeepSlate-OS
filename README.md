# DeepSlate-OS
A graphical operating system build for CC: Tweaked that uses Basalt for its GUI.

## How to install:
Simply run `wget https://raw.githubusercontent.com/Fireshoa/DeepSlate-OS/refs/heads/main/install.lua install.lua` and then run `install`, and your computer will install DeepSlateOS. You can also enter the shell and run install again to factory reset your os. If you dont want this, run these commands first:
`copy settings settingsbackup`
`copy apps appsbackup`
and restore with
`copy settingsbackup settings`
`copy appsbackup apps`

## Features:
 - Build in app browser
   - Shows you every app in your `apps/` directory and lets you run them
 - Run
   - Like the app browser, but has you type the app ID. Lets you run hidden apps.
 - Customization
   - Lets you change the colors of your system :)
 - Text Editor
   - While it currently does not support highlighting, it will in the future, don't worry!
